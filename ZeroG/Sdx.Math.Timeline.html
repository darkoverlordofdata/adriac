Description:

public class Timeline : Tweenbase

A Timeline can be used to create complex animations made of sequences and parallel sets of Tweens.

The following example will create an animation sequence composed of 5 parts:

  1. First, opacity and scale are set to 0 (with Tween.set() calls).
  2. Then, opacity and scale are animated in parallel.
  3. Then, the animation is paused for 1s.
  4. Then, position is animated to x=100.
  5. Then, rotation is animated to 360Â°.

This animation will be repeated 5 times, with a 500ms delay between each iteration:

Timeline.CreateSequence()
    .Push(Tween.Set(myObject, OPACITY).Target({ 0 }))
    .Push(Tween.Set(myObject, SCALE).Target({ 0, 0 }))
    .BeginParallel()
         .Push(Tween.To(myObject, OPACITY, 0.5f).Target({ 1 }).Ease(Interpolation.QuadInOut))
         .Push(Tween.To(myObject, SCALE, 0.5f).Target({ 1, 1 }).Ease(Interpolation.QuadInOut))
    .End()
    .PushPause(1.0f)
    .Push(Tween.To(myObject, POSITION_X, 0.5f).Target({ 100 }).Ease(Interpolation.QuadInOut))
    .Push(Tween.To(myObject, ROTATION, 0.5f).Target({ 360 }).Ease(Interpolation.QuadInOut))
    .Repeat(5, 0.5f)
    .Start(myManager);

Static methods:

  • public static Timeline CreateSequence ()
    Creates a new timeline with a 'sequence' behavior. Its children will be delayed so that they are triggered one after the other.
  • public static Timeline CreateParallel ()
    Creates a new timeline with a 'parallel' behavior. Its children will be triggered all at once.

Methods:

  • protected void Setup (TimelineModes mode)
  • public Timeline Push (Tween tween)
    Adds a Tween to the current timeline. Nests a Timeline in the current one.
  • public Timeline PushPause (float time)
    Adds a pause to the timeline. The pause may be negative if you want to overlap the preceding and following children.
  • public Timeline BeginSequence ()
    Starts a nested timeline with a 'sequence' behavior. Don't forget to call End to close this nested timeline.
  • public Timeline BeginParallel ()
    Starts a nested timeline with a 'parallel' behavior. Don't forget to call End to close this nested timeline.
  • public Timeline End ()
    Closes the last nested timeline.
  • public GenericArray<Tweenbase> GetChildren ()
    Gets a list of the timeline children. If the timeline is started, the list will be immutable.
