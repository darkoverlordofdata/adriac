Entitas

An ECS (entity component system) based on: Entitas-CSharp

Content:

Namespaces:

  • Event - Lifecycle events:

Classes:

  • System - Systems provide a convenient way to group systems. All systems will be initialized and executed based on the order you added them.
  • Group - Use world.GetGroup(matcher) to get a group of entities which match the specified matcher. Calling world.GetGroup(matcher) with the same matcher will always return the same instance of the group. The created group is managed by the world and will always be up to date. It will automatically add entities that match the matcher or remove entities as soon as they don't match the matcher anymore.
  • Matcher - Match entities by component complile list of components to bit array for fast comparison
  • World - A world manages the lifecycle of entities and groups. You can create and destroy entities and get groups of entities.
