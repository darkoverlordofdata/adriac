Mark

Description:

public static Tween Mark ()

Convenience method to create an empty tween. Such object is only useful when placed inside animation sequences (see Timeline), in which it may act as a beacon, so you can set a callback on it in order to trigger some action at the right moment.

Returns:

The generated Tween.
