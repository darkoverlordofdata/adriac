GetValues

Description:

public TweenAccessorGetValues GetValues

Gets one or many values from the target object associated to the given tween type. It is used by the Tween Engine to determine starting values.

Parameters:

returnValues

An array which should be modified by this method.

target

The target object of the tween.

tweenType

An integer representing the tween type.

Returns:

The count of modified slots from the returnValues array.
